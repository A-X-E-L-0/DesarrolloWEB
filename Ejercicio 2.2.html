2.2 Complete las siguientes Oraciones

  1. El elemento hr inserta una regla horizontal
  2. Para marcar una superficie se utiliza el elemento sup y para marcar un subindice se utiliza el elemento sub
  3. El elemento de encabezado menos importante es h6 y el elemendo de encabezado mas importante es h1
  4. El elemento ul marca una lista desordenada
  5. El elemento p marca un parrafo
  6. El atributo type="reset" en un elemento input inserta un botón que, cuando se presiona, restablece el contenido del formulario
  7. El elemento tr marca una fila de una tabla.
  8. Los asteriscos se usan por lo general como caracteres de enmascarado en un cuadro de contraseña.
